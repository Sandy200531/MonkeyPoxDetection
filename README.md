# 🐒 Monkeypox Detection using Modified VGG16 & Custom CNN

## 📌 Overview
This project is a deep learning-based image classification system for detecting **Monkeypox** from skin lesion images. It compares the performance of a **Modified VGG16 Transfer Learning Model** with a **Custom CNN** model.

The application is built using **Python**, **TensorFlow/Keras**, **OpenCV**, and **Tkinter** for the graphical user interface.

---

## ✨ Features

- Upload Monkeypox image dataset
- Image preprocessing and normalization
- Train Modified VGG16 model
- Train Custom CNN model
- Compare model performance
- Display Accuracy, Precision, Recall, F1-Score, and AUC
- Confusion Matrix visualization
- Predict Monkeypox from a test image
- User-friendly Tkinter GUI

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Sandy200531/MonkeyPoxDetection.git
```

Move into the project

```bash
cd MonkeyPoxDetection
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the virtual environment

Windows

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python MonkeyDetection.py
```
